# Change Log

All notable changes to the "opener" extension will be documented in this file.

## [0.0.1] - 2024-11-06

### Added

- Initial release
- Preset action: Open iOS folders in Xcode
- Preset action: Open Android folders in Android Studio
- Preset action: Open folders in current VSCode window
- Preset action: Open folders in new VSCode window
- Custom folder actions configuration support
- Error handling and user notifications
- Output channel logging for debugging
- Context menu integration with folder-only visibility

### Features

- Right-click context menu actions for folders
- Configuration-driven custom actions
- Shell command execution with proper error handling
- Detailed logging to "Opener" output channel

## [Unreleased]

### Planned
